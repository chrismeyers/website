# web

### Run development server
```
$ npm run dev
```

### Run production preview
```
$ npm run build && npm run preview
```

### Run tests
```
$ npm test
```

### Lint and fix files
```
$ npm run lint
$ npm run lint:fix
```

## Setup
1. Create `.env.development.local` and add the following variables:
    - `VITE_LIGHTGALLERY_LICENSE`
