# web

### Run development server
```
$ pnpm dev
```

### Run production preview
```
$ pnpm build && pnpm preview
```

### Run tests
```
$ pnpm test
```

### Lint and fix files
```
$ pnpm lint
$ pnpm lint:fix
```

## Setup
1. Create `.env.development.local` and add the following variables:
    - `VITE_LIGHTGALLERY_LICENSE`
